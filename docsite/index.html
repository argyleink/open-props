Getting Started

Modern CSS and PostCSS versions available.

Installation

Grab the props with a URL or from NPM.

NPM

npm install open-props
CSS
Bundles

                  @import "open-props";
                  @import "open-props/reset.css";
                                  

                  import 'open-props';
                  import 'open-props/reset.css';
Individual packs

                  @import "open-props/sizes.css";
                  @import "open-props/colors.css";
                  @import "open-props/gradients.css";
                  /* etc */
                

                  import 'open-props/sizes.css';
                  import 'open-props/colors.css';
                  import 'open-props/gradients.css';
                  /* etc */
PostCSS JIT Props

Ship only what you use! Learn more 

npm install postcss-jit-props
Props from CSS

                  require('postcss-jit-props')(
                    files: [
                      require('open-props/props.css'),
                    ]
                  )
Props as Javascript

                  require('postcss-jit-props')(
                    require('open-props/props.js')
                  )

Animations

Premade keyframe effects for slides and fades allow you to orchestrate your own combinations. Plus, a few nice attention grabbers and indeterminate state animations.

The Props

              --animation-fade-{in,out}
              --animation-slide-out-{up,down,left,right}
              --animation-slide-in-{up,down,left,right}
              --animation-shake-{x,y}

              --animation-spin: op-spin 2s linear infinite;
              --animation-ping: op-ping 5s var(--ease-out-3) infinite;
              --animation-blink: op-blink 1s var(--ease-out-3) infinite;
              --animation-float: op-float 3s var(--ease-inout-3) infinite;
              --animation-bounce: op-bounce 2s var(--ease-squish-2) infinite;
              --animation-pulse: op-pulse 2s var(--ease-out-3) infinite;
Keyframe Examples

              @keyframes op-fade-in {
                to { opacity: 1 }
              }

              @keyframes op-slide-out-right {
                to { transform: translateX(100%) }
              }

              @keyframes op-slide-in-up {
                from { transform: translateY(100%) }
              }

              @keyframes op-float {
                50% {
                  transform: translateY(-25%);
                }
              }

            }
