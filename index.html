Getting Started

Modern CSS and PostCSS versions available.

Installation

Grab the props with a URL or from NPM.

NPM

npm install open-props

Colors

Open Props includes Open Color (Version 1.9.1): an open-source color scheme. Optimized for UI.

Open devtools, open the color picker, set the swatches to the page's custom properties, and enjoy picking from the set!
